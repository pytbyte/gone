Karo

School Managment System

Karo is a Cloud based School Management System designed to offer Simplified, Reliable and Efficient solutions through modern technology. Schools can register, setup and manage data from any internet enabled device with no installation or setup cost and operations run remotely with on-demand and Automated resources delivered on the dashboard and via sms or email to Parent/Gurdian .




Students

Admin & Class teacher

Create, Read, Update & Delete Student data

Manage student Attendance records



Staff

Admin

Create,Read ,Update and delete Staff data

Assign Staff Roles (class teacher etc)



Accademics

Admin & Class Teacher

Create class data (class, stream etc)

Create,Read ,Update and delete Accademic records

Generate Report Forms


Payments

Parent, Admin & Staff

Parents can make payments from bank or mobile.

Admin can manage staff payments data and make payment to staffs' bank or mobile account manually or automatically at set intervals



Performance

System, Class teacher

Automated Data passed from accademic records and visualised to clarify students accademic progress over time.

Class teacher can update performance data and add remarks.


Reports

System & Admin

System Creates and sends notifications to users(Staff,Admin,Parent) upon request completion (reciepts etc).

Admin can create and send custom sms or email notifications

Admin can authorize report sending for upcoming events
